Integration of an Undo/Redo button in the Apple iOS notes app

The goal of this project is to intuitively integrate an undo/redo button in the Apple Notes app. Users don’t seem to really enjoy the current shake to undo so I proposed three alternative solutions.

User research
Solutions
1. Bottom nav bar

The bottom navigation bar is a decent position to input the undo icon because it will be easy to reach and remember. However, the bottom nav bar already has five icons which is the most a nav bar can hold as per iOS Material Design principles.To create space for an undo icon I integrated a kebab menu icon to hold the Delete note and New note icons.

2. The Long-press

Shaking the iphone is the set shortcut to undo/redo but maybe this shortcut isn’t intuitive enough. An alternative shortcut would be a swipe left to undo and swipe right to redo.
